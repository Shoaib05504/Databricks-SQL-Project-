# Databricks SQL Project - Airbnb Dataset

## Project Overview
This project demonstrates the use of SQL in Databricks to perform data analysis using DDL (Data Definition Language) and DML (Data Manipulation Language) operations on an Airbnb dataset.

The goal is to analyze listing prices, neighbourhood trends, and perform data manipulation for better insights.

---

## Tools Used
- Databricks
- SQL

---

## Dataset Description
The dataset contains Airbnb listings information including:

- Name – Listing name  
- Neighbourhood – Location of the property  
- Price – Cost per listing  
- Property Type – Type of accommodation  
- Review Scores Rating – Customer rating  

---

## DDL Operations Performed
- High Rated Listings (Review score > 85)
- Mid Price Range Listings (Price between 100 and 300)
- Price Categorization using CASE statement
- Grouping by Neighbourhood
- Sorting Listings by Price

---

## DML Operations Performed
- Insert new records into dataset
- Update existing records
- Perform aggregation (GROUP BY)
- Filter high-price listings
- Count total listings

---

## Project Steps
1. Loaded and previewed dataset  
2. Applied DDL queries to filter and structure data  
3. Performed DML operations for data manipulation  
4. Executed aggregation and filtering queries  
5. Created visualizations for price comparison  

---

## Visualizations
- Average price by neighbourhood
- Comparison of listing prices across locations

---

## Output
- All SQL queries are executed in Databricks  
- Screenshots of outputs are included in the repository  
- Reports are available as notebooks (Report 1 & Report 2)

---

## Conclusion
This project helped in understanding practical usage of SQL in Databricks, including data filtering, transformation, grouping, and visualization.

It also demonstrates how real-world datasets can be analyzed to extract meaningful insights.
